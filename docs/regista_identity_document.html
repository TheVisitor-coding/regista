Direction créative
Le beau jeu,
entre tes mains.
Regista incarne l'art du football intelligent. Pas le football des statistiques froides — celui de la vision, du timing, du plaisir de jouer juste. L'identité visuelle doit donner envie de jouer dès le premier regard : colorée, vivante, arrondie, mais avec la profondeur d'un vrai jeu de stratégie.

Ton éditorial — comment Regista parle
Direct Enthousiaste Accessible Encourageant Jargon FM Froid / technique Trop formel
Le jeu parle comme un fan de foot passionné, pas comme une base de données. Victoire → "Magnifique !" pas "Match gagné". Transfert réussi → "Tu as frappé fort sur ce coup."
Règles design à ne jamais violer
  • DO Utiliser les gradients sur les fonds de cards et headers — jamais sur les textes
  • DO Border-radius minimum 10px sur tous les composants — pas d'angles droits
  • DO Vert #2E9E5B pour toutes les actions principales (CTA, statuts positifs)
  • DO Orange #FF6B35 réservé aux actions urgentes (blessure, suspension, alerte)
  • NON Jamais de couleur rouge seule pour un état critique — trop alarmiste pour un jeu
  • NON Pas de texte blanc pur (#FFF) — utiliser le blanc chaud #EAE8E1
  • NON Pas de grilles sans respiration — padding minimum 16px dans chaque card
